# Bad Drink

This site is a quiz site that recommends a terrible drink based on answers to ridiculous questions.

## Run Locally

1. fork and clone this repo.
2. run npm i
3. run npm start
4. visit localhost:3000

## Build

* HTML5
* CSS3/SASS
* Javascript
